# Cookido Agent

Cookidoo recipe management agent - automates recipe export/import and organization on cookidoo.es.

## Installation

```bash
uv sync
```

## Usage

```bash
uvx cookido-agent
```

## Configuration

Set environment variables (via `.envrc.local`):
- `COOKIDOO_USERNAME`: Cookidoo account email
- `COOKIDOO_PASSWORD`: Cookidoo account password
